# 💻 Tecnologías Web · HTML, CSS, XML & JavaScript

Conjunto de prácticas de la asignatura **Tecnologías Web** (3º curso) que recorren las tecnologías fundamentales del frontend: desde el modelado de datos con **XML/XSD** hasta sitios web completos con **HTML5**, **CSS3** y validación con **JavaScript**.

## 📚 Contenido

### 🔹 01 · XML, DTD y XSD
Modelado y validación de documentos XML mediante **DTD** y **XML Schema (XSD)**. Cinco ejercicios que cubren distintos casos prácticos (biblioteca, etc.).

### 🔹 02 · XHTML y XSL
Transformación de documentos XML a HTML usando **XSLT** y documentos **XHTML** estrictos.

### 🔹 03 · Sitio Web completo
Sitio multi-página con **HTML5** y **CSS3**:
- `index.html` — página principal
- `productos.html` — catálogo
- `noticias.html` — sección de novedades
- `contacto.html` — formulario de contacto
- `guia.html` — guía/manual
- Hojas de estilo personalizadas y carpeta de imágenes

### 🔹 04 · JavaScript y validación
Validación de formularios en cliente con **JavaScript** y manipulación dinámica del DOM en página de productos.

## 🎯 Conceptos aplicados

- Estructuración semántica con HTML5
- Diseño y estilos con CSS3
- Modelado de datos con XML, DTD y XML Schema
- Transformaciones XSLT
- Validación de formularios en el lado del cliente
- Manipulación del DOM con JavaScript

## 🚀 Cómo visualizar

Abre cualquier archivo `.html` directamente en un navegador, o sirve la carpeta con un servidor local:

```bash
# Python 3
python -m http.server 8000
```
